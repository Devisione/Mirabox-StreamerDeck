# StreamDock Node.js Plugins

Ready-to-use plugins for streamers.

## Debug Interface

A web-based debug interface is available for all plugins at:

```
http://localhost:23519/
```

This interface allows you to:
- View plugin logs in real-time
- Monitor WebSocket connections
- Track plugin events and state
- Debug plugin behavior

Open `http://localhost:23519/` in your browser while StreamDock is running to access the debug interface.
